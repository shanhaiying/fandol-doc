歧义错误

该处是歧义错误的类别清单。下面列出了不同原因的报错和错误输出。

S.No 类 报错信息 报错原因
1 e_des ! LaTeX Error: There's no line here to end 在description环境长标签后使用\\
2 e_center ! LaTeX Error: There's no line here to end 在center环境首行后使用\\
3 e_foot  ! Argument of \@sect has an extra } 在\section中使用弱命令
4 e_ragged  ! Argument of \@caption has an extra } \raggedright或\raggedleft环境中使用\\
5 e_and  ! Extra alignment tab has been changed to \cr table，array或eqnarray.环境中&过多
6 e_cline  ! Extra alignment tab has been changed to \cr \cline的参考栏不存在
7 e_col  ! Extra alignment tab has been changed to \cr. tabular*环境中使用@
8 e_num  ! Missing number treated as zero \\后使用非数参数
9 e_asterisk Missing * at the end of the line \\后*未与括号使用
10 e_pbox_miss  ! Missing number, treated as zero. \parbox[t]{} ..参数丢失
11 e_mis_circle  ! Missing number, treated as zero. \circle丢失数参数
12 e_list  ! Argument of \lst@next has an extra } 弱命令\parbox中使用1stlisting
13 e_capacity  ! TeX capacity exceeded, sorry [input stack size=1500] 弱命令\parbox中使用1stlisting
14 e_runaway Runaway argument? 括号丢失，如 \cline{1-2 替换为\cline{1-2}
15 e_verbatim Runaway argument? 在其他命令中使用抄录，例如\ifthenelse
16 e_undefined ! Undefined control sequence 未知命令
17 e_footnote ! Undefined control sequence \footnote中使用\footnote
18 e_integral  ! Missing { inserted. 配对越界
19 e_zeta  ! Missing { inserted. 下标错误
20 e_bezier  ! Illegal unit of measure (pt inserted). \qbezier丢失数值参数
21 e_too_bezier  ! Illegal unit of measure (pt inserted). \qbezier选项过多
22 e_unit  ! Illegal unit of measure (pt inserted) \parbox[t]{2} ..第二参数单位非法
23 e_symfoot  ! LaTeX Error: Counter too large. 使用symbolic脚注时大于9个
24 e_large_count  ! LaTeX Error: Counter too large. 计数器值>26
25 e_begin ! LaTeX Error: Missing \begin{document} \begin{document}前或\begin{document}后存在文本
26 e_margin  ! LaTeX Error: Missing \begin{document}. \marginsize误用

一般错误

S.No 类 报错信息 报错原因
1 e_fileEnd ! File ended while scanning use of \end. 一般为括号丢失引起
2 e_end No message only an asterisk, i.e * \end{document}丢失
3 e_illegal LaTeX Error: Illegal character in array arg 使用r,l和c之外的字符
4 e_tab ! Misplaced alignment tab character & 丢失\begin{tabular}
5 e_backslash ! Missing \endcsname inserted 环境名中使用反斜线，例如\begin{\itemize}

7 e_right ! Extra \right \right未与\left配对 ，或 \end{array}丢失

10 e_parameter ! Illegal parameter number in definition of... \newcommand参数过多，\newcommand{\test}[1]{#3}
11 e_cmd ! LaTeX Error: Command ... already defined 试图定义已存在命令，\newcommand{\time}

19 e_equation  ! Display math should end with $$ 等式中使用$$

24 e_sty  ! LaTeX Error: File `anysize1.sty' not found 宏包不可用
25 e_doc_class  ! LaTeX Error: Can be used only in preamble. \documentclass脱离导言区

33 e_lost  ! LaTeX Error: Float(s) lost. 未定义计数器
34 e_lonely  ! LaTeX Error: Lonely \item--perhaps a missing list environment. 在list环境外使用\item
35 e_parg  ! LaTeX Error: Missing p-arg in array arg. tabular环境中丢失p参数
36 e_hash  ! You can't use `macro parameter character #' in vertical mode. 一般模式下使用#
37 e_enlarge  ! LaTeX Error: Suggested extra height (14454.0pt) dangerously large. \enlargethispage给出数值过大
38 e_deftab  ! LaTeX Error: Undefined tab position. 未定义tabbing

46 e_twoclass  ! LaTeX Error: Two \documentclass or \documentstyle commands. 多文档类声明

48 e_superscript  ! Double superscript. 双上标，例如2^3^4
49 e_clash_opt  ! LaTeX Error: Option clash for package csvtools. 同宏包选项冲突
50 e_unknown_opt  ! LaTeX Error: Unknown option ... for package ... 未定义宏包选项
51 e_hyphenation  ! Improper \hyphenation will be flushed. \hyphenation的参数不当
52 e_stack_size  ! TeX capacity exceeded, sorry [main memory size=1000000] 溢出，命令定义错误
53 e_environment  ! LaTeX Error: Environment ... undefined. 未定义环境
54 e_midline  ! LaTeX Error: \< in mid line 命令 \< 应置于行开始处
55 e_infinite Goes into infinite loop \\strut\hrule用法
