排版与输入不同，TeX也不同于其他的文字处理工具。本文谈及一些需要特殊主义的TeX的格式问题。

1. 文本

  • TeX中，词后和句后应放置空格，放置多少空格都可以，甚至是100个。TeX将输入文件中的行结束视作空格。段落使用空行分割。 
  • TeX会自动处理行尾的连字号与文字分割，所以不必以连字号分断文字。否则会在行中出现连字符如 "author-prepared copy。"
  • 引号需输入为成对的左引号和右引号，如 quoted text； 不要使用双引号("bad form")。
  • 不要使用下划线。在排版中，强调的文本会使用变换字体实现，通常是slanted 或者italic字体。
  • 一些普通字符被解释为命令，因此，如果想用到正文中，需要在之前放置反斜线（\）： $ & % # { 和 } 需输入为 \$ \& \% \# \{ 和 \}。
  • 不推荐使用增加垂直或水平距离。注意多使用标记命令处理文档的内容和细节，而不是担心排印出的页面。

2. 数学

正文中的数学式需使用$， 例如$\pi r^2$ 生成 r2。如想改变上下标为Roman字体，需使用\rm命令，如$J_{\rm HF}(t)$生成JHF(t)。

特显公式可以使用多种方式输出。参见有关特显公式标记的用户手册。

如果作者可以定义自己的公式编号，令LaTeX自动生成编号并不烦难。默认情况下，LaTeX会按照通篇文章来进行编号。

3. 交叉引用

文本中的公式，表格和图像的交叉引用依赖于“keys”的使用，这些需由用户定义。\label命令用来定义LaTeX中交叉引用的keys， \ref命令用来关联keys。 Keys是普通的文本串，用来标记灯饰，表格和图像，故它们可以在文本中像符号一样被关联。对于节和类似的，放置\label命令于标记命令之后来开启关联结构。 对于图像和表格标题，置\label命令于\caption，\tablecaption或\figcaption，例如：

    \tablecaption{This is a caption.\label{tab1}}

不要再你的论文中关联页码。

LaTeX会在当前工作目录下生成含有引用关联的辅助文件，扩展名为 .aux。该文件不能删除，LaTeX会处理辅助文件中的数据来生成引用。

辅助文件机制会使运行LaTeX多次便可确保交叉引用正常工作。当有关数字发生变更时，会改变等式，表格和其他类型的编号。LaTeX会自动生成警告信息来建议用户更新交叉引用。
